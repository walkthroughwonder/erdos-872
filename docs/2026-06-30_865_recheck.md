# 2026-06-30 #865 recheck

Verdict: still NOT-RELEVANT to the #872 attack ([2026-06-26 #865 check](2026-06-26_865_solution_check.md), [arXiv:2606.29361](https://arxiv.org/abs/2606.29361)). The new arXiv abstract for 2606.29361 defines \(f_3(N)\) as a pairwise-sums threshold for sets \(A \subseteq \{1,\ldots,N\}\), proves \(f_3(N) \le 5N/8+O(1)\), and combines this with the standard \([N/8,N/4] \cup [N/2,N]\) construction to resolve Erdős Problem 865 ([arXiv:2606.29361](https://arxiv.org/abs/2606.29361)).

That confirms the June 26 verdict rather than changing it ([2026-06-26 #865 check](2026-06-26_865_solution_check.md), [arXiv:2606.29361](https://arxiv.org/abs/2606.29361)). The #865 mechanism is additive combinatorics for pairwise sums, while #872 is the divisibility primitive-set saturation game formalized in PR #4226 ([2026-06-26 #865 check](2026-06-26_865_solution_check.md), [formal-conjectures PR #4226](https://github.com/google-deepmind/formal-conjectures/pull/4226)). The arXiv abstract adds process relevance because it says an earlier conditional version was formalized in Lean 4 with no sorries and no added axioms, but it does not mention divisibility antichains, Maker-Breaker saturation, T2 reachability, q-fiber residual games, or safe-edge obstructions ([arXiv:2606.29361](https://arxiv.org/abs/2606.29361)).
